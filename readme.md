### pyDispSys
pyDispSys is a synth interface
for PureData Synths / Sequencers / Drums
it Communicates with a PureData over a OSC UDP
connection and draws simple graphics with
the pygame library.

### Requires
pyOSC
pygame
320x240 LCD display
LCD on /dev/fb1

### Controls
Esc - quit
1 - 3 grid size
s - synth
d - drumz
q - seuqencer
